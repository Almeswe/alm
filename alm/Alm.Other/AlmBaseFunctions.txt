alm language base functions:

	print(string message) of integer   -> prints message in console;
	println(string message) of integer -> prints message in console with "\n" special symbol (new line);
	input() of string				   -> reads line from console and returns this line;

	tostr(integer num) of string -> converts integer argument to string;
	tostrf(float numf) of string -> converts float argument to string;

	toint(string str) of integer -> converts string argument to integer, if it's possible;
	tofloat(string str) of float -> converts string argument to float, if it's possible;

	point(integer num) of float   -> convert integer argument to float;
	round(float numf)  of integer -> convert float argument to integer (can occure data losses);
